MyStudyBible 소개

오랫동안 안드로이드 스마트폰을 사용하면서 MySword 라는 앱을 오랫동안 사용해왔습니다.
그러다가 아이폰으로 바꾸게 되었고, 아이폰에서는 제 취향에 맞는 앱이 없다는 것을 알게되었습니다.
그래서 기존에 사용하던 앱의 데이터도 그대로 사용하고 싶은 마음을 담아서 새롭게 아이폰용 앱을 만들게 되었습니다.
아래에 소개하는 성경, 주석, 지도 데이터베이스 형식은 MySword 앱에서 사용하던 데이터베이스 형식과 동일하므로 데이터는 호환될겁니다.
특히 MySword 에서 사용하던 사용자 노트가 담겨있는 sqlite 파일을 불러와서 텍스트를 추출해 MyStudyBible 앱의 노트에 추가해주는 기능도 넣어두었습니다.
다만 주석(관련구절) 데이터베이스 형식은 theWord 라는 PC앱의 데이터 형식과 호환되는 형식입니다.
반드시 MySword 나 theWord 용 데이터 뿐만 아니라 아래의 형식에 맞기만 하면 읽어와서 사용할수 있으므로 데이터베이스 파일의 구조를 이곳에 남겨두겠습니다.
누구라도 아래와 같은 형식의 데이터를 만들수 있다면 이 앱을 통해서 성경공부를 할 수 있습니다.
또한 성경 읽기 계획표는 기본으로 제공되는 csv 파일을 참조 및 수정해서 자신만의 성경 읽기 계획표를 만들어 사용하실수 있도록 하였습니다.

시작하기: 폴더 구조 및 파일 추가


앱을 처음 실행하면 '나의 iPhone' 저장 공간에 MyStudyBible 폴더가 자동으로 생성되며, 그 아래에 다음과 같은 7개의 하위 폴더가 만들어집니다. 각 폴더에 해당 형식의 파일을 복사해 넣으면 앱에서 인식하여 기능을 사용할 수 있습니다.

  • bible: Sqlite 형식의 성경(예: *.bbl.mybible) 파일을 이곳에 넣으세요.
  • commentary: Sqlite 형식의 주석(예: *.cmt.mybible) 파일을 이곳에 넣으세요.
  • cross_ref: Sqlite 형식의 관주(예: *.xrf.twm) 파일을 이곳에 넣으세요.
  • font: 사용하길 원하는 .ttf 또는 .otf 폰트 파일을 이곳에 넣으세요.
  • Images: 앱 내에서 사진 접근 권한을 허용하고 선택한 사진이 자동으로 복사되는 곳입니다. 또는, '책이름-장번호-절번호_' 형식으로 시작하는 이미지 파일을 직접 추가할 수도 있습니다.
  • mapdata: Sqlite 형식의 지도 데이터 파일을 이곳에 넣으세요.
  • user_data: 앱 내에서 생성된 사용자 노트, 하이라이트, 북마크, 불러온 읽기 계획표 등의 사용자 데이터가 sqlite 형식으로 저장되는 곳입니다. (직접 파일을 넣는 폴더가 아닙니다)


성경 데이터

성경 구절을 저장하는 테이블 구조입니다.
CREATE TABLE "Bible" (
                "Book" INT,
                "Chapter" INT,
                "Verse" INT,
                "Scripture" TEXT
            )

주석 데이터

성경 구절에 대한 주석 정보를 저장하는 테이블 구조입니다.
CREATE TABLE "commentary" (
                "id" INTEGER,
                "book" INTEGER,
                "chapter" INTEGER,
                "fromverse" INTEGER,
                "toverse" INTEGER,
                "data" TEXT
            )

관주 데이터

성경 구절 간의 참조 관계를 저장하는 테이블 및 뷰 구조입니다.
            
            CREATE TABLE xrefs(
                id integer primary key, 
                vi1 int, 
                vi2 int
            )
            
            
            CREATE VIEW xrefs_bcv as 
                select 
                    id, 
                    (vi1 & 2130706432)>>24 fbi, 
                    (vi1 & 16711680)>>16 fci, 
                    (vi1 & 65280)>>8 fvi, 
                    (vi1 & 255) fspan, 
                    (vi2 & 2130706432)>>24 tbi, 
                    (vi2 & 16711680)>>16 tci, 
                    (vi2 & 65280)>>8 tvi, 
                    (vi2 & 255) tspan 
                from xrefs

지도 데이터

성경 속 지명과 위치 정보를 저장하는 테이블 구조입니다.
            
            CREATE TABLE verse(
                id INTEGER PRIMARY KEY, 
                book INTEGER, 
                chapter INTEGER,

            CREATE TABLE location(
                id INTEGER PRIMARY KEY,

주요 기능

sqlite 형식의 성경파일 지원

인터넷에서 구할수 있는 sqlite 데이터베이스 형식으로 구성된 성경 파일들을 지원합니다.

하이라이트 & 노트

중요한 구절을 하이라이트하고 각 구절마다 개인 노트를 추가하여 성경 공부를 더욱 깊게 할 수 있습니다.

sqlite 형식의 지도파일 지원

sqlite 형식의 지도 데이터 파일을 지원하여 구절마다 해당 구절에 있는 지명을 간단히 지도에서 확인할수 있습니다.

사진 첨부 기능

해당 구절에 대해 이해에 도움이 되는 사진을 첨부해서 더욱 쉽게 성경을 이해할수 있습니다.(예: 성전 사진, 제사장 의복 사진 등)

강력한 검색

성경 검색 뿐만 아니라, 사용자가 입력한 구절별 노트의 내용도 검색할수 있습니다.

읽기 계획

제공된 읽기 계획 뿐만아니라, 사용자가 직접 만든 다양한 성경 읽기 계획으로 꾸준히 말씀을 가까이할 수 있습니다.

성경 해설

인터넷에서 구할수 있는 sqlite 데이터베이스 형식으로 된 주석서들을 통해 각 구절에 대한 해설과 주석으로 성경의 배경과 의미를 더 잘 이해할 수 있습니다.
