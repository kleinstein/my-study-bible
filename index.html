MyStudyBible 소개

아래와 같은 형식의 데이터베이스 파일들을 지원하는 성경앱입니다.


지원하는 Sqlite 데이터베이스 형식:


성경 데이터

성경 구절을 저장하는 테이블 구조입니다.
CREATE TABLE "Bible" (
                "Book" INT,
                "Chapter" INT,
                "Verse" INT,
                "Scripture" TEXT
            )

주석 데이터

성경 구절에 대한 주석 정보를 저장하는 테이블 구조입니다.
CREATE TABLE "commentary" (
                "id" INTEGER,
                "book" INTEGER,
                "chapter" INTEGER,
                "fromverse" INTEGER,
                "toverse" INTEGER,
                "data" TEXT
            )

관주 데이터

성경 구절 간의 참조 관계를 저장하는 테이블 및 뷰 구조입니다.
CREATE TABLE xrefs(
                id integer primary key, 
                vi1 int, 
                vi2 int
            )

            CREATE VIEW xrefs_bcv as 
                select 
                    id, 
                    (vi1 & 2130706432)>>24 fbi, 
                    (vi1 & 16711680)>>16 fci, 
                    (vi1 & 65280)>>8 fvi, 
                    (vi1 & 255) fspan, 
                    (vi2 & 2130706432)>>24 tbi, 
                    (vi2 & 16711680)>>16 tci, 
                    (vi2 & 65280)>>8 tvi, 
                    (vi2 & 255) tspan 
                from xrefs

지도 데이터

성경 속 지명과 위치 정보를 저장하는 테이블 구조입니다.
CREATE TABLE verse(
                id INTEGER PRIMARY KEY, 
                book INTEGER, 
                chapter INTEGER, 
                verse INTEGER, 
                location TEXT collate nocase
            )

            CREATE TABLE location(
                id INTEGER PRIMARY KEY, 
                location TEXT collate nocase, 
                root TEXT collate nocase, 
                locinfo TEXT, 
                lat DOUBLE, 
                lon DOUBLE, 
                verses TEXT collate nocase, 
                comment TEXT collate nocase
            )

주요 기능

sqlite 형식의 성경파일 지원

인터넷에서 구할수 있는 sqlite 데이터베이스 형식으로 구성된 성경 파일들을 지원합니다.

하이라이트 & 노트

중요한 구절을 하이라이트하고 각 구절마다 개인 노트를 추가하여 성경 공부를 더욱 깊게 할 수 있습니다.

sqlite 형식의 지도파일 지원

sqlite 형식의 지도 데이터 파일을 지원하여 구절마다 해당 구절에 있는 지명을 간단히 지도에서 확인할수 있습니다.

사진 첨부 기능

해당 구절에 대해 이해에 도움이 되는 사진을 첨부해서 더욱 쉽게 성경을 이해할수 있습니다.(예: 성전 사진, 제사장 의복 사진 등)

강력한 검색

성경 검색 뿐만 아니라, 사용자가 입력한 구절별 노트의 내용도 검색할수 있습니다.

읽기 계획

제공된 읽기 계획 뿐만아니라, 사용자가 직접 만든 다양한 성경 읽기 계획으로 꾸준히 말씀을 가까이할 수 있습니다.

성경 해설

인터넷에서 구할수 있는 sqlite 데이터베이스 형식으로 된 주석서들을 통해 각 구절에 대한 해설과 주석으로 성경의 배경과 의미를 더 잘 이해할 수 있습니다.
